The `InputWithIcon` component is a flexible compound component for building input fields with an icon positioned on either the left or right side. It is composed of `InputWithIcon`, `InputWithIconIcon`, and `InputWithIconInput` sub components.

<Alert variant="info" className="not-prose my-5">
  <Info />
  <AlertTitle>We use a slot to pass the icon as a child.</AlertTitle>
  <AlertDescription>
    Passing icons as props requires a client component. Using{" "}
    <code>react-slot</code> allows us to pass the icon as a child instead.
  </AlertDescription>
</Alert>

## Usage

```tsx
<InputWithIcon>
  <InputWithIconIconSlot>
    <Search />
  </InputWithIconIconSlot>
  <InputWithIconInput placeholder="Search..." />
</InputWithIcon>
```
