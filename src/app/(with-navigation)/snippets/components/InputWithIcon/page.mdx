The `InputWithIcon` component is a customizable input field that supports icons on either the left or right side. It's built on top of the base `Input` component and provides a clean way to add icons to your input fields while maintaining proper spacing and alignment.

Key features:

- Supports icons on both left and right sides via the `iconXPosition` prop
- Maintains proper spacing and alignment of the icon
- Fully customizable through className props for the container, input, and icon
- Inherits all native input props through ComponentPropsWithoutRef

## Examples

### Default

<InputWithIcon icon={Search} inputProps={{ placeholder: "search..." }} />

### Right icon

<InputWithIcon
  icon={Search}
  iconXPosition="right"
  inputProps={{ placeholder: "search..." }}
/>

### Right icon with RTL

<InputWithIcon
  icon={Search}
  className="[direction:rtl]"
  iconXPosition="right"
  inputProps={{ placeholder: "جستجو..." }}
/>

### Left icon with RTL

<InputWithIcon
  icon={Search}
  className="[direction:rtl]"
  iconXPosition="left"
  inputProps={{ placeholder: "جستجو..." }}
/>
