# Next js 16 Portfolio with custom shadcn registry

A modern, open-source portfolio website built with Next.js 16, React 19, and TypeScript. Features a collection of reusable components, hooks, and utilities that can be installed via a custom registry system.

## ‚ú® Features

- **Portfolio Showcase**: Display your projects, skills, and contact information
- **Code Snippets**: Organized collection of reusable code snippets with MDX support
- **Bookmarks**: Curated collection of useful resources and links
- **Component Registry**: Custom registry system for sharing components, hooks, and utilities
- **Dark/Light Theme**: Built-in theme switching with `next-themes`
- **SEO Optimized**: Comprehensive metadata, structured data, and sitemap generation
- **Type-Safe**: Full TypeScript support with typed routes
- **Modern Stack**: Built with the latest Next.js 16 features including Cache Components and React Compiler

## üöÄ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) with App Router
- **UI Library**: [React 19](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) with custom design system
- **Components**: [shadcn/ui](https://ui.shadcn.com/) (New York style)
- **Content**: [MDX](https://mdxjs.com/) with `next-mdx-remote`
- **Code Highlighting**: [Shiki](https://shiki.matsu.io/)
- **Linting/Formatting**: [Biome](https://biomejs.dev/)
- **Package Manager**: [Bun](https://bun.sh/)

## üìù Contributing

Contributions are welcome! Please feel free to submit a Pull Request. For major changes, please open an issue first to discuss what you would like to change.

1. Clone the repository
2. install dependencies: 
   ```bash
   bun install
   ```
3. Set up environment variables
   create a `.env` file
   ```env
   NEXT_PUBLIC_FRONTEND_DOMAIN=https://yourdomain.com
   ```
4. Run the development server
   ```bash
   bun run dev
   ```
