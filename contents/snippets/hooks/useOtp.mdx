as of today, its on [limited availability](https://developer.mozilla.org/en-US/docs/Web/API/OTPCredential) but you can add it as progressive enhancement noting would happen if browser doesn't support it.

you need to follow this sms format in order for this work:

```text
Your verification code is 123456.

@domain #code
```

for example for my website,it would be like this:

```text
Your verification code is 123456.

@mhl5.vercel.app #123456
```

## Usage

```tsx
import { useOtp } from "@/hooks/useOtp";
```

<br />

```tsx
useOtp((code) => {
  form.setFieldValue("otp", code);
});
```
