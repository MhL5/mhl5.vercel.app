## Usage

### Basic Usage

```tsx
const [value, setValue] = useUrlState("name");
```

### With default value

You can pass a value or a function to the `defaultValue` option to set the state initial value.

```tsx
const [value, setValue] = useUrlState("name", {
  defaultValue: () => "default value",
});
```

### With history replace

```tsx
const [value, setValue] = useUrlState("name", {
  history: "replace",
});
```
