`useMediaQuery` returns `false` on the server (SSR) and the actual match result on the client. It accepts CSS media query strings with auto-complete support. 

## Usage
  
```ts
import { useMediaQuery } from "@/hooks/useMediaQuery";
```

<br/>

```ts
const isMobile = useMediaQuery("(min-width: 640px)");
```
